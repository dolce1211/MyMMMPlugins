## コマンド入力は windows 環境で Powershell 上で動作することを前提とする

- 以下のようなコマンドは実行できません。(&&は使えません)

```
cd packages\cdk\layers\plantuml-layer\opt\plantuml && echo "現在のplantuml.jarのサイズ:" && dir plantuml.jar
```

- 代わりに PowerShell コマンドを使用してください。

```
cd packages\cdk\layers\plantuml-layer\opt\plantuml; dir plantuml.jar
```
